# Task 0 — Scaffolding a Next.js Project (ALX Project 0x00)

## Objective
This task demonstrates how to scaffold a new Next.js project using the official CLI tool `create-next-app`.  
It focuses on understanding how to initialize a modern React + TypeScript setup with Tailwind CSS and ESLint.

---
